Существует 3 популярных цикла

2. Цикл while 
Выполняется если условие только true;
а) Бесконечный цикл. Пока условие true;
б) С увеличивающимся инкрементом, с уменьшающимся инкриментом
в) С ложным условием
г) с не равным 0
д) сравнение с if
у) ввод через консоль

3. Цикл do while
Срабатывает хотя бы один раз. 
Сначала выполняет потом проверяет условие

4. Цикл for
a) конечное условие завершения цикла
б) переменная i видна только внутри условия for
b) можно в условии использовать все числовые переменные
