1. Синтаксис написания циклов также ничем не отличаются от написания в других языках программирования.
   Оператор while.

let i = 0;

while (i < 3) {

  console.log("i = " + i);

  i++;

}

2.цикл for
for (let i = 0; i < 3; i++) {

  console.log("i = " + i);

}

3. Создание функций

function showMessage() {

  alert( 'Всем привет!' );

}

showMessage();

Далее мы вызываем функцию, так как объявив функции но не вызвав ее мы не увидим результат.

4. Локальные и внешние переменные
Переменные, объявленные внутри функции, видны только внутри этой функции(локальные переменные).
У функции есть доступ как к локальным так и внешним(объявленные снаружи) переменным.
let school = 'Bitlab';
function showMessage() {

  let message = school + ' helped me to learn coding!';

  alert(message);

}

showMessage();

5. Параметры
Мы можем передать внутрь функции любую информацию, используя параметры (также называемые аргументами функции).

function showMessage(from, text) {

  alert('message from ' + from + ': ' + text);

}

showMessage('Yedil', 'Hi!');

showMessage('Kuat', "How are you?");
