Bootstrap - это свободный фреймворк, разработанный Twitter, 
который позволяет разработчикам быстро и легко создавать красивые и отзывчивые веб-сайты и приложения.
Bootstrap предоставляет набор инструментов для создания сетки, стилей, компонентов интерфейса и многого другого.

В целом, Bootstrap - это мощный и гибкий инструмент для создания красивых и отзывчивых веб-сайтов и приложений. 
Он может быть использован как начальная точка для вашего проекта, либо как полноценный фреймворк для разработки.

14.layout
деление на 12 колонок
row и col-*-* являются основными классами Bootstrap для создания сетки. 
Они позволяют создавать резиновые и отзывчивые макеты, адаптирующиеся под различные разрешения экрана.
 вы можете использовать числа от 1 до 12. Число указывает количество столбцов, которые будет занимать данный элемент в ряду.
