DOM (объектная модель документа)

Document Object Model, 
сокращённо DOM – объектная модель документа,
которая представляет все содержимое страницы в виде объектов, 
которые можно менять. 

DOM (Document Object Model) в JavaScript — это программный интерфейс, позволяющий программам читать и
изменять содержимое веб-страницы, структурируя его как объектное представление документа.
Это представление включает в себя структуру документа, стиль и содержимое, 
позволяя взаимодействовать с текстом, стилями и структурой HTML через JavaScript.


Объект document – основная «входная точка».
С его помощью мы можем что-то создавать или менять на странице.

DOM представляет HTML-документ в виде структуры, 
где каждый элемент, атрибут, текстовое содержимое и другие узлы являются объектами, 
которые могут быть манипулированы с помощью JavaScript.

1. вызов функции с помощью метода onclick
<input type="button" value="Click" onclick="show()">

function show(){
        alert("HELLO!")
    }

2. вызов функции через id 
<input type="button" value="Click" id="element">
<script>
    function show(){
        alert("HELLO!");
    }
    element.onclick=show;
</script>


есть минусы вызова данной функции так - он не будет вызывать два раза ее, допустим:
вызовет только последнюю.

5. value внутри инпута. Пример с изменением цвета.

<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        div{
            width: 100px;
            height: 100px;
            border-radius: 50%;
            border: 2px solid black;
            margin-top: 10px;
        }
    </style>
</head>
<body>
<input type="text" id ="colorChange">
<button id = "btn">Change color</button>
<div id ="element"></div>
<script>
    function change(){
       let color = colorChange.value;
       element.style.background = color;
    }

    btn.addEventListener('click', change);
</script>
</body>
</html>
