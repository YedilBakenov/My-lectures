1. Класс начинается со слова abstract

2.1. Классы должны наследоваться от абстрактного главного класса через слово extends.

2.2. Абстрактный класс не может создавать свои экземпляры. 
Это общая сущность для всех. Нам не нужны его объекты.

2.3. Абстрактный класс - это сущность.            

4. абстрактный класс может иметь методы как обычные так и абстрактные

5. абстрактные методы нужно обязательно переписать в каждом классе, которые наследуются
