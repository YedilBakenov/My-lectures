Наследование - это механизм, при котором один класс приобретает свойство другого класса.
Главный класс называется супер классом, а наследующий класс - подклассом.

1. Extends class
2. Наследуем переменные, геттеры сеттеры, методы
3. Конструктор в наследуемых классах
	super(name, age);
	this.klichka = klichka;
4. Запуск метода от супер класса через обьект подкласса
5. Создать массив через главный класс и добавить туда объекты наследуемых классов
6. Под полиморфизмом  в Java подразумевается то, что программе необязательно знать какого именно типа будет объект, главное что он потомок главного класса.
7. Создание пустого обьетка и присвание его любому подклассу
8. instanceOf() - является конкретный объект указанным классом
